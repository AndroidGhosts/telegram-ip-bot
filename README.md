# telegram-ip-bot
# بوت تيليجرام للحصول على نطاقات IP ومعلومات CDN

هذا البوت يساعدك في الحصول على نطاقات IP لأي نطاق مع معلومات CDN الخاصة بكل نطاق.

## كيفية التشغيل

1. أضف التوكن في الإعدادات (Secrets) باسم `TELEGRAM_BOT_TOKEN`.
2. البوت سيعمل تلقائيًا عبر GitHub Actions.

## الأوامر

- أرسل نطاقًا مثل `google.com` لتحليله.
- `/asn [رقم ASN]` للبحث عن نطاقات ASN معينة.
- `/cdn [نطاق IP]` للتحقق من مزود CDN.
- `/cdns` لعرض قائمة مزودي CDN المعروفين.
